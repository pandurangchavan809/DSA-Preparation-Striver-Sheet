================ DSA C++ RULES =================

#include<bits/stdc++.h>    :- this include all the liabraries c++ have... it takes bit more time but that's ok.

1) Compile a single C++ file
   g++ filename.cpp -o filename

2) Run executable (PowerShell)
   .\filename

3) Compile with warnings (recommended)
   g++ filename.cpp -Wall -o filename

4) Compile & run (one line)
   g++ filename.cpp -o filename ; .\filename

5) Multiple C++ files
   g++ main.cpp helper.cpp -o main

6) Input from file
   .\main < input.txt

7) Output to file
   .\main > output.txt

8) Compile with C++17 standard
   g++ filename.cpp -std=gnu++17 -o filename

9) Debug build
   g++ filename.cpp -g -o filename

10) Clean executable
   del filename.exe
